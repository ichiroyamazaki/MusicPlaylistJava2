# MusicPlaylist.java

A Java program that simulates a **music playlist manager**, demonstrating how to store, organize, and interact with songs using object‑oriented programming concepts.  
This file showcases practical use of classes, lists, and methods to build a simple playlist system.

## Features
- Create and manage a playlist of songs  
- Add, remove, and search for tracks  
- Display playlist contents in order  
- Demonstrates **ArrayList** usage for dynamic storage  
- Well‑commented code for easy understanding  

## Usage
Compile and run:
```bash
javac MusicPlaylist.java
java MusicPlaylist
